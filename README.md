## üöÄ Deployment

To inaugurate the Astoria system in your environment:

```bash
# Clone the repository
git clone https://github.com/QamerHassan/Smart-Hotel-Management-Platform.git

# Enter the estate
cd Smart-Hotel-Management-Platform

# Ignite the engines
docker-compose up --build
```

Access the **Guest Portal** at `http://localhost:3000` and the **Manager's Terminal** at `http://localhost:3000/login`.
